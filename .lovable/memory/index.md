# Project Memory

## Core
- Tech Stack: iOS and watchOS targets using Capacitor for native deployment.
- Apple Ecosystem: Full widget support (iPhone Home/Lock screen, Apple Watch complications).
- Design: Minimalist UI inspired by iBreathe with breathing-like pulse animations.
- Accessibility: High priority on color contrast, large font sizes, and legibility.
- Pomodoro Logic: 25m focus, 5m short break. 30m long break after 4 focus sessions.
